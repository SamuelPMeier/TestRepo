Let us assume we need to sort the elements of an array of alphabetic characters in descending order, ie. from ‘A’ to ‘Z’. We can assume the array begins with an element at index 0, a common practice in computer science applications. Using the Insertion Sort algorithm, we begin by designating the element at index 0 in the array sorted, because a single element array is already sorted. The element at index 0 is ‘Y’. Looking ahead to index 1, we find the letter ‘L’. We swap ‘L’ for ‘Y’ at index 1; index 0 has the element ‘L’ as a result of the swap. Now index 0 and index 1 are in the sorted part of the array.  We continue the process of swapping elements when necessary, and remembering the largest index of the sorted part of the array, until the largest index sorted reaches the length of the array.  

--Insertion Sort Facts--

*Insertion sort functions optimally on lists with lengths between 5 and 10 elements.

*Insertion sort performs poorly on lists with a reverse order.

--Example of insertion sort in python--

def insertionSort(alist):

   for index in range(1 , len(alist) ):

     currentvalue = alist[index]

     position = index

     while position > 0 and alist[position - 1] > currentvalue:

         Alist[position] = alist[position - 1]

         position = position - 1

     Alist[position] = currentvalue

alist = ['A', 'G', 'B', 'D', 'C', 'F', 'E']

insertionSort(alist)

print(alist)
