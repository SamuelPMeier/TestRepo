Insertion sort sorts the elements of a list or array in a loop. Each pass through the Insertion Sort loop, an element, 'element a', of the list is removed, and element a is replaced in a new position according to an arbitrary set of sorting rules, ordered in a descending or ascending fashion.

The sorted part of the list begins with the first element, because a one element list is already sorted. Each successive iteration of the insertion sort loop compares the next element to be sorted with the elements in the sorted portion of the list, and selects the right position in the sorted part of the list for the next element to be inserted. If the next element to be sorted is found to have a value greater than the any element in the sorted part of the list, in an ascending insertion sort, the next element becomes the last element in the sorted list.

Insertion Sort Facts:

*Insertion sort functions optimally on lists with lengths between 5 and 10 elements.

*Insertion sort performs poorly on lists with a reverse order.
