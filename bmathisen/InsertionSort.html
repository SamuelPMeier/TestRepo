Insertion Sort Technical Description

The Insertion sort traverses through an array, picking up the elements as it goes along and inserts

them into a sorted list by comparing the picked up element against the sorted arrays elements.

If the element is larger than the compared to element it leaves the element in place. If the element

is smaller it finds the correct position by comparison and shifts all the larger elements up in the

array to make space within the array and inserts the picked up element into the newly created position.
