Insertion Sort Technical Description


Given an unsorted array, an insertion sort algorithm will sort the elements of said array in ascending order.
This algorithm traverses each index of the unsorted array with a for loop, starting with the second index.
In each iteration of the for loop, the element in the array at the index being traversed is stored as the ‘current’ element.
Then, a while loop continuously swaps the ‘current’ element with the element before it in the array until either:

  1. The element directly before the ‘current’ element is smaller than the ‘current’ element
  2. The ‘current’ element is at index 0

Either of these conditions causes the while loop to stop and the next iteration of the for loop to start, until all indices have been iterated through.
