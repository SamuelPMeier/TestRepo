Insertion Sort

Insertion sort is a sorting algorithm that sorts a sequence of elements.

Input:  

A sequence of n elements  {e1,  e2, .… ,  en}.

Output:  

A permutation {e’1, e’2, … , e’n} of the input sequence such that  e’1 ≤ e’2 ≤ …  ≤ e’n

Steps:

  1. Have an array of unsorted elements
  2. Set a marker for the sorted section after the first elements in the list
  3. Repeat steps A through C until the unsorted section is empty

A.           Select the first unsorted element

B.           Swap this elements to the left until it arrives at the correct sorted position

C.           Advance the marker to the right by one position

Insertion Sort Example:

Pseudocode of the Insertion Sort Algorithm (array is zero-based):

for i ← 1 to length(A)
   j ← i
   while j > 0 and A[j-1] > A[j]
       swap A[j] and A[j-1]
       j ← j - 1
   end while
end for
