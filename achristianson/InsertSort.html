Insertion sort method is a simple sorting algorithm used to sort elements in small arrays and lists. The algorithm iterates by consuming one element from a array or list, finding the position for the consumed element in a sorted list by comparing its value with that of other elements, and inserting the consumed element in the correct position within the sorted list.

 

This is normally done by creating the sorted list at the front of the array. At the beginning this will consist of the first element. At each array position afterward, the next consumed element is compared with the greatest value of the top sorted element. If the top sorted element is less than the consumed element, the consumed element is left in place. If the top sorted element is greater than the current consumed element, the greater sorted element is shifted upward and the consumed element is compared with the next top sorted element.
